# Start

1. Download
```sh
mkdir clickhouse
cd clickhouse
curl https://clickhouse.com | sh
```

This downloads a binary

2. Run server
```sh
./clickhouse server
```

3. Run client
```sh 
./clickhouse client
```

4. Connect to cloud
```sh
./clickhouse client --host lfunr6iu75.eu-central-1.aws.clickhouse.cloud --secure --password 'p1sw0rd'
```

# Client
1. Databases and tables
```sql
SHOW DATABASES
SHOW TABLES FROM default
SHOW TABLES FROM system
```
2. Queries
```sql
SELECT price
FROM s3('https://datasets-documentation.s3.eu-west-3.amazonaws.com/uk-house-prices/parquet/house_prices_all.parquet')
ORDER BY -date ASC
LIMIT 10
```
